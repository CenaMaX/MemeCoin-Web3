
# Meme Coin - Web3 Project


## Run Locally

Clone the project

```bash
  git clone [https://github.com/CenaMaX/MemeCoin-Web3](https://github.com/CenaMaX/MemeCoin-Web3.git)
```

Go to the project directory

```bash
  cd client
```

run the project

```bash
  npm run dev
```
